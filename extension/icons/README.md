# Icons

Place PNG icons here before submitting to any browser store.

| File | Size | Used for |
|------|------|----------|
| `icon16.png`  | 16×16   | Browser toolbar (small), favicon |
| `icon32.png`  | 32×32   | Windows taskbar, retina toolbar |
| `icon48.png`  | 48×48   | Extensions management page |
| `icon128.png` | 128×128 | Store listing, install dialog |

All four sizes are required. Design at 128×128 and scale down, or provide separate
artwork optimized for small sizes.
